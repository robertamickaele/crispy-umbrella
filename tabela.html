Tags HTML

Tags Funcionalidade
header Define um cabeçalho.
html Esta tag define que o documento tem um formato HTML.
button Define um botão.
nav Define um conjunto de links de navegação.
style Esta tag é utilizada para estilizar um elemento.
title Define o título da página.
body Define o corpo do documento HTML.
p Define um parágrafo.
a Define um hyperlink.
h1 Define um elemento de cabeçalho de maior hierarquia.
h2 Define um elemento de cabeçalho de hierarquia menor que h1.
ul Cria uma lista sem ordem específica.
li Define uma lista.
ol Cria uma lista ordenada.
div Define uma divisão ou uma seção de elementos num documento HTML.
